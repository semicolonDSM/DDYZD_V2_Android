# DDYZD_V2_Android
동아리 지원을 도와주는 Android app
## Activities
* ChatList
    * 채팅리스트를 보여주는 Activity
* ChattingPage
    * 채팅 Activity
* ClubDetails
    * 동아리 상세정보를 보여주는 Activity
* LoginActivity
    * Login을 위한 Activity
## Fragments
* ChooseModifyDialog
    * 수정선택을 위한 dialog
* ClubList
    * 동아리 리스트를 보여주는 fragment
* GitSheetDialog
    * 깃허브정보를 수정하는 dialog
* MainFeed
    * 동아리 게시물을 보여주는 fragment
* ModifySheet
    * 소개를 수정하는 dialog
* MyPage
    * 내정보를 보는 fragment
* BottomSheetDialog
    * 피드삭제dialog
* NotSheetDialog
    * 권한없음을 보여주는 dialog
* UserInfo
    * 동아리원정보를 보여주는 fragment
